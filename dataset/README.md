# 数据集目录结构

请按照 **动作 / 视角 / 被试** 的三层结构组织视频文件。

```
dataset/
├── 1.原地旋转/
│   ├── front/              # 正面拍摄
│   │   ├── subject_01/     # 被试编号
│   │   │   ├── trial_01.MOV
│   │   │   └── trial_02.MOV
│   │   └── subject_02/
│   └── side/               # 侧面拍摄
│       ├── subject_01/
│       └── subject_02/
├── 2.小推车/
│   ├── front/
│   └── side/
├── 3.原地纵跳/
│   ├── front/
│   └── side/
├── 4.前滚翻/
│   ├── front/
│   └── side/
├── 5.超人飞/
│   ├── front/
│   └── side/
└── 6.直线加速跑/
    ├── front/
    └── side/
```

## 拍摄要求

- **固定三脚架**，不要手持跟拍
- 每段视频仅包含**一次动作**
- 视频开头/结尾可以有站立等待等噪声，系统会自动裁剪
- 被试编号可自行命名（如 `subject_A`、`小明` 等），保持一致即可
- 如果某个动作/视角组合没有视频，留空目录即可

## 视角说明

| 视角 | 适合评估的指标 |
|------|-------------|
| **front**（正面） | 左右对称性、侧向摆动、双侧协调 |
| **side**（侧面） | 躯干倾角、跳跃高度、膝屈角、前后运动 |

## 文件格式

支持 `.MOV`、`.mp4`、`.m4v`、`.avi` 等常见视频格式。
