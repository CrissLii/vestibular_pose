# 🧠 儿童统感训练量化评估规范（LLM结构化版）

---

## 📌 一、通用定义

### 坐标系

| 轴 | 含义                 |
| - | ------------------ |
| X | 左右方向               |
| Y | 垂直方向               |
| Z | 前后方向（若无深度，则使用平面投影） |

---

### 关键点定义

| 名称           | 定义                            |
| ------------ | ----------------------------- |
| nose         | 鼻子关键点                         |
| mid_shoulder | (L_shoulder + R_shoulder) / 2 |
| mid_hip      | (L_hip + R_hip) / 2           |

---

### 重心代理

```text
COG ≈ mid_hip
```
